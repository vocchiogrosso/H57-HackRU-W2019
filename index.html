About

Modular Sense is a Modular Robot that has built in componets that allows it to be useful in a variety of stituations.

What we've done is the robot from scratch that can support integrating all diffrent types of modular sensors and other boards.

How It Works

One application of the Modular Sense is in the use of a Farming Bot.

The robots modular capability allows us to add componenets and integrate into it's existing platform.

For example, you could connect your camera to the onboard ardunio and remotely run your code

The robot is capable for running approx. 4 minutes on 5000mah battery

A larger battery can be used, 10000mah, giving you almost 6 minutes.
